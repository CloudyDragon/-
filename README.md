# 这是次元界的前端界面

## 技术栈
> vue2 + vuex + vue-router + webpack + ES6

## 运行项目
> cnpm run dev //如果你是npm的话你要去修改package.json文件里面的cnpm为npm
## 发布项目
> cnpm run build //这里打包后的html文件有问题。js引入部分没有引号。等我写完后台在处理
